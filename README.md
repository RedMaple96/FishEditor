# Fish Path Editor (鱼游路径编辑器)

这是一个基于 Web 技术开发的 2D 路径编辑器，专为规划和生成平滑的物理路径（如鱼的游动轨迹）而设计。
用户可以通过自由手绘或点击布置坐标点的方式创建路径，程序会自动将路径平滑化（基于 Bezier 曲线），并生成每一个节点的位置 (X, Y) 以及它的旋转角度 (Angle)。

## ✨ 核心特性

- **多模式绘制**：支持“自由画线”与“放置坐标点”两种模式。自由画线后会自动通过 Ramer-Douglas-Peucker 算法抽稀并生成平滑的关键点。
- **完美的物理平滑**：内置全局的 Bezier Spline 平滑算法，生成极其自然流畅的 C1 连续物理曲线，杜绝生硬的折角。
- **自定义坐标系与分辨率**：可自由调整分辨率（默认 1280x720），并采用**左下角**为坐标原点 `(0, 0)` 的数学系坐标标准（X轴向右为正，Y轴向上为正）。
- **无极缩放与平移**：支持通过鼠标滚轮进行无极缩放（0.1x 到 10x），支持鼠标右键或中键拖拽平移画布，方便编辑超长路径或进行微调。
- **动态坐标点重采样**：
  - **全局增加**：可以基于当前的平滑曲线形态，全局均匀地插入指定数量的坐标点，且不改变原有的物理轨迹弧度。
  - **局部区域调整**：支持在画布上选中任意两点（A点和B点），在它们之间增加或减少坐标点。
- **路径镜像与方向控制**：支持一键将路径进行水平镜像、垂直镜像，或完全反转鱼的游动方向（头尾对调）。
- **多路径导入与游动模拟**：
  - 支持批量上传已生成的 `.dat` 文件，可在画布中以半透明线条区分并同时显示多条路径。
  - 支持为每条导入的路径独立控制游动播放与速度（1~100 级实时调节），进行多条鱼同时游动的群游效果模拟测试。
- **强大的真实鱼资源渲染 (PixiJS/Spine)**：
  - **批量导入**：支持批量导入多类型的真实鱼资源（支持单图 `.png/.jpg`、序列帧 `.plist + .png`、Spine 动画 `.json + .atlas + .png`）。
  - **独立装配**：可以为主路径和每一条导入的附加路径单独分配不同的真实鱼资源进行模拟，系统会自动缓存并在 Canvas 上层以 WebGL 引擎实时渲染对应的鱼动画。
  - **物理朝向校正**：鱼资源在游动时会严格根据轨迹的数学切线方向实时计算旋转角度，支持向左游动时的自动镜像翻转，确保“头始终在前、背始终朝上”。
- **完善的撤销重做系统**：每一次有效编辑都会记录在历史栈中，支持顶部按钮快速撤销重做。
- **双模数据导出**：
  - 导出标准的 `.dat` 文件，数据格式为精简的嵌套数组 `[[x, y, angle], ...]`。
  - 支持同步导出带有透明背景的 `.png` 画布截图留底。
  - 支持直接复制数据到剪贴板。

## 🚀 快速开始

### 前置要求
- [Node.js](https://nodejs.org/) (建议 v16+)
- npm 或 pnpm

### 安装与运行

```bash
# 1. 进入项目目录
cd FishEditor

# 2. 安装依赖
npm install

# 3. 启动开发服务器 (本地)
npm run dev

# (可选) 使用预设的 Shell 脚本以后台进程方式启动公网 HTTPS 服务
chmod +x start.sh stop.sh
./start.sh
```

> **注意**：本项目已配置 Vite 使用 `@vitejs/plugin-basic-ssl` 插件，默认开启 HTTPS 并绑定 `0.0.0.0` 暴露到局域网/公网，默认端口为 `5178`（端口可能随占用情况递增）。

## 🛠️ 技术栈

- **框架**: [React 18](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **构建工具**: [Vite](https://vitejs.dev/)
- **样式**: [Tailwind CSS](https://tailwindcss.com/)
- **渲染层**: HTML5 Canvas 2D API (轨迹绘制) + [PixiJS](https://pixijs.com/) & Pixi-Spine (真实鱼动画渲染)
- **核心算法**:
  - Ramer-Douglas-Peucker 算法 (手绘轨迹抽稀)
  - 贝塞尔样条曲线算法 (Bezier Spline Interpolation)
  - 弧长参数化算法 (Arc-Length Parameterization)

## 📖 使用指南

1. **调整分辨率**：点击顶部导航栏的 **画布** 按钮展开面板，设置目标屏幕分辨率，画布中央的红框即为分辨率边界（原点在左下角）。
2. **导入鱼资源（可选）**：
   - 点击顶部导航栏的 **鱼资源** 按钮展开面板。
   - 点击上传区域，多选上传你的美术资产（如果是 Spine 或序列帧，请将配套的图片、图集、数据文件一并选中上传）。
3. **绘制主路径**：
   - 选中右上角的 **自由画线**，在画布上按住鼠标左键拖动，松开后自动生成点。
   - 选中右上角的 **坐标点**，在画布上依次点击，生成精确折点，程序会自动计算出平滑过渡。
4. **调整视图**：滚动鼠标滚轮可缩放画布，按住鼠标右键可拖动平移视图。
5. **调整路径密度与局部重采样**：
   - 展开左侧 **绘制与调整** 面板。
   - 使用“全局均匀增加”或点击“选点 A/B”进行“局部区域调整”，改变坐标点密度。
6. **主路径模拟测试**：
   - 在左侧 **绘制与调整** 面板底部找到“模拟游动”。
   - 在“模拟鱼资源”下拉框中选择刚刚导入的鱼动画（或保持“默认形状”）。
   - 调整鱼大小与游动速度，点击“开始模拟”查看鱼沿路径游动的真实效果。
7. **多路径导入与群游测试**：
   - 点击顶部导航栏的 **导入多路径** 按钮展开面板。
   - 点击上传按钮，可多选批量上传本地的 `.dat` 路径文件。
   - 导入后每条路径都会分配一种区别颜色并显示在列表中，您可以为每条路径**独立指定不同的鱼资源**，并**独立调节**其“游动速度”。
   - 点击列表头部的全局“播放”按钮，在右侧画布观察它们群游的动态效果。
8. **导出数据**：展开左下角的 **导出路径数据**，输入文件名，点击相应的导出按钮下载 `.dat`（仅数据）或 `.png`（带截图）。

## 📄 导出的数据格式

导出的 `.dat` 文件实质上是一个 JSON 格式的嵌套数组。每一项代表一个点，保留两位小数。

格式：`[ [X坐标, Y坐标, 旋转角度], ... ]`

示例：
```json
[
  [-235.94, 480.50, 20.89],
  [494.64, 201.70, 42.32],
  [514.99, -203.28, 87.12]
]
```
> **角度说明**：由于坐标系为左下角原点，这里的角度代表了该点切线的绝对方向角度（逆时针为负，范围在 -180 ~ 180 度之间），可直接应用于物理引擎中的物体旋转（如鱼头朝向）。
